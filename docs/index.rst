Joseki Documentation
====================

**Date**: |today| | **Version**: |version|

Reference atmospheric thermophysical profiles for radiative transfer applications in Earth's atmosphere.

If you are new to Joseki, start by reading the :doc:`"Usage" <usage>` section.
The :doc:`reference <reference/index>` provides the detailed specification of the public API.

Installation
------------

You can install Joseki via `pip <https://pip.pypa.io/en/stable/>`_ from `PyPI <https://pypi.org/>`_:

.. code-block:: shell

   pip install joseki

or via `conda <https://docs.conda.io>`_ from `conda-forge <https://conda-forge.org>`_:

.. code-block:: shell

   conda install -c conda-forge joseki
